# Soundboard

## Description:  
This is a simple soundboard application designed for Microsoft Teams, featuring a collection of funny meme sounds. With this tool, you can quickly play popular meme audio clips during your Teams meetings to add humor and fun to your conversations. The soundboard is easy to use and customizable, making it perfect for lightening the mood in virtual meetings or online hangouts.
